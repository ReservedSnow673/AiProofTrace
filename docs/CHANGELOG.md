# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-19

### Added
- Initial release of AiProofTrace.
- ProofTraceRegistry smart contract for on-chain anchoring.
- Express server for inference recording and verification.
- CLI tool for end-to-end auditability.
- Multi-chain support for Base and Sepolia.
- Comprehensive test suite for contracts and server.
- Threat model and security documentation.
